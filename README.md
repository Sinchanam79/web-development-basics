# Web Development Basics

This repository documents my foundational learning in web development.
It focuses on building strong fundamentals required for scalable and
maintainable frontend applications.

## Topics Covered
- HTML: semantic structure, forms, accessibility basics
- CSS: selectors, box model, Flexbox, responsive layouts
- JavaScript: fundamentals, DOM manipulation, events, basic logic.

## Purpose
This repository reflects my consistent practice and understanding of
core web development concepts. It serves as a foundation for advanced
frontend frameworks and open-source contributions.

## Repository Structure
The repository is organized in a chapter-wise manner with practice
exercises, assignments, and mini projects to reinforce core concepts.
